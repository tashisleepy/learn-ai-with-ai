## How to Use This Course

1. **Open** ChatGPT, Claude, or Gemini
2. **Copy-paste** the prompts from each week
3. **Read** the AI's response
4. **Run** the code in Google Colab
5. **Ask** follow-up questions if confused
6. **Repeat**

### The Golden Rule

If you are confused, ask the AI:

```
Explain this like I am 15 years old.
Use examples from YouTube, Instagram, or video games.
No technical words.
```
